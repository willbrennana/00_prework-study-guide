Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The meta element contains information about the meta data.
  • The body element represents the visible content shown to the user.
  • The paragraph element represents a block of text.
  • The three list elements represent unordered lists, ordered lists, and list items.
  • The image element contains information about images that are displayed on the webpage.
  • The break element creates a line of empty space, or a line break, between two blocks of content.
  • The anchor element creates links to the same webpage or other webpages.
