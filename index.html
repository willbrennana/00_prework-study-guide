HTML

  • HTML is the standard markup language for Web pages.
  • The head element contains information about the webpage.
  • The meta element contains information about the meta data.
  • The body element represents the visible content shown to the user.
  • The paragraph element represents a block of text.
  • The three list elements represent unordered lists, ordered lists, and list items.
  • The image element contains information about images displayed on the webpage.
  • The break element creates a line break between two blocks of content.
  • The anchor element creates links to the same webpage or other webpages.

CSS

  • Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML.
  • A margin indicates how much space we want around the outside of an element.
  • A border indicates how much space we want between the margin and padding.
  • A padding indicates how much space we want around the content inside an element.
  • A content indicates how much space we want for the featured information.
  • A declaration contains two important components: the CSS property we want to apply and the value of the property.
  • A property indicates a feature (i.e. color), while a value assigns a descriptor to a property (i.e. blue).

Git

  • Git is free and open source software for distributed version control.
  • git status checks and identifies current branch.
  • git checkout -b branch-name creates a new branch and switches to it.
  • git commit saves changes made.
  • git pull opens a pull request.
  • git push pushes up all changes.

JavaScript

  • JavaScript is a programming language that adds interactivity to your website.
  • A variable is a named container that allows us to store data in our code.
  • The set of types in the JavaScript language consists of primitive values and objects.
  • Chrome DevTools is a set of web developer tools built directly into the Google Chrome browser.
  • Control flow is the order in which a computer executes code in a script.
  • Conditional statements and loops control the order in which code is executed, or if it is even executed at all.
  • An array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
  • A function is a set of instructions that tells the computer how to perform a certain task.
